# focus-log

フォーカスセッションを記録・管理するアプリ。

## 技術スタック

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- ESLint / Prettier

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 で起動します。

## コマンド

```bash
npm run dev      # 開発サーバー起動
npm run build    # 本番ビルド
npm run lint     # ESLint
npm run format   # Prettier フォーマット
```
